Change Password

Hier können Sie ihr Password ändern. Später finden Sie diese Option im Bereich System > Administration.

General Section

Geben Sie hier Ihre Basisinformationen an. Sie können Sie später im Bereich Freifunk > Basic Settings und Freifunk > Contact ändern.

Radio 0 Mesh Section

Nehmen Sie hier die Einstellungen vor mit denen sich ihr Router mit dem Freifunknetz verbinden kann. Ihr Gerät und benachbarte Freifunk Knoten müssen auf demselben Kanal senden. Je nach Gerätetyp können Sie zwischen verschiedenen 2,4Ghz und 5Ghz Kanälen auswählen. Hier gibt es eine Übersicht auf welchem Kanälen und in welchem Mode ihre Community sendet.

{Hilfe Distance Optimization}

Ihre Mesh IP Adresse erhalten Sie von der Freifunk Gemeinschaft in Ihrer Nachbarschaft. Es ist eine netzweit eindeutige Identifikation, z.B. 104.1.1.1.

Access Point Configuration

Hier können Sie den Access Point für Clients ihres Freifunknetzwerkes konfigurieren. Dazu können Sie einen IP-Addressraum von Ihrer Community bekommen und hier eintragen. Ein Netzwerkrechner hilft beim berechnen der richtigen Netzwerkaddresse und des CIDR-Suffix.

Beachten Sie beim Vergeben des Netzwerknamens (z.B. meinname.freifunk.net) daruf, dass Sie unterschiedliche Namen vergeben, falls Sie mehrere Router in Ihrer Umgebegung betreiben, da ein Client mit seiner IP nicht die Access Points wechsen kann.

Wenn die Felder leer bleiben wird ein Netzwerk automatisch nach den vorgaben aus dem Feld Freifunk Comunity erstellt und der Netzwerkname berechnet sich aus AP+Kanal+SSID der Freifunk Community,

Falls Sie keinen Access Point für Ihre Nachbarn konfigurieren möchten überspringen Sie einfach diesen Punkt.
